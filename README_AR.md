# تطبيق فاتورة الغاز

تطبيق React + Tailwind CSS بامتداد JSX، يعمل باللغة العربية ويدعم الوضع الفاتح والداكن والحفظ على الجهاز والعمل كتطبيق PWA على أندرويد.

## المميزات

- حساب فواتير المنازل حسب 3 شرائح.
- حساب التجاري (معادل منازل).
- حساب المخابز البلدية بسعر موحد قابل للتعديل.
- توزيع الاستهلاك على عدد شهور المحاسبة قبل تطبيق الشرائح.
- إضافة القسط إلى إجمالي الفاتورة.
- حفظ الفواتير في `localStorage`.
- بحث وفلترة وحذف فاتورة أو حذف السجل بالكامل.
- نسخة احتياطية بصيغة JSON واسترجاعها.
- تعديل أسعار الشرائح وتاريخ التحديث.
- Dark / Light Mode.
- تصميم Responsive للموبايل والكمبيوتر.
- PWA قابلة للتثبيت على أندرويد وتعمل بعد تحميلها مرة واحدة حتى عند ضعف الاتصال.

> ملاحظة: الأسعار الافتراضية مأخوذة من الصور المرجعية: 6 و8 و12 جنيه، وسعر المخابز 0.141. راجع الأسعار الرسمية وعدّلها من صفحة «أسعار الشرائح».

## تشغيل المشروع في VS Code

افتح Terminal داخل مجلد المشروع وشغّل:

```bash
npm install
npm run dev
```

سيظهر رابط محلي غالباً مثل:

```text
http://localhost:5173
```

## إنشاء نسخة Production

```bash
npm run build
npm run preview
```

مجلد النشر الناتج هو:

```text
dist
```

## رفع المشروع على GitHub Pages

المشروع يحتوي بالفعل على الملف:

```text
.github/workflows/deploy.yml
```

1. أنشئ Repository جديداً على GitHub.
2. ارفع جميع ملفات المشروع إلى الفرع `main`.
3. افتح `Settings` ثم `Pages`.
4. في `Build and deployment` اختر `GitHub Actions`.
5. افتح تبويب `Actions` وانتظر اكتمال عملية النشر باللون الأخضر.
6. سيظهر رابط التطبيق في صفحة `Settings > Pages`.

لا تحتاج لتعديل `base` لأن المشروع يستخدم `base: "./"` و`HashRouter` لتجنب مشكلة الصفحة البيضاء بعد النشر.

## تثبيت التطبيق على أندرويد

بعد رفعه على رابط HTTPS:

1. افتح الرابط على الهاتف باستخدام Google Chrome.
2. اضغط قائمة الثلاث نقاط.
3. اختر `Install app` أو `إضافة إلى الشاشة الرئيسية`.
4. وافق على التثبيت.
5. سيظهر التطبيق بأيقونته ويعمل في نافذة مستقلة مثل التطبيقات العادية.

قد يظهر زر «تثبيت الآن» داخل التطبيق تلقائياً عندما يسمح المتصفح بذلك.

## مكان حفظ البيانات

الفواتير والأسعار تحفظ داخل متصفح الجهاز باستخدام `localStorage`، وليست على سيرفر. لا تمسح بيانات Chrome قبل تنزيل نسخة احتياطية من صفحة الإعدادات.

## أهم الملفات

```text

src/pages/HistoryPage.jsx       السجل والحذف
src/pages/RatesPage.jsx         تعديل الأسعار
src/pages/SettingsPage.jsx      الوضع والنسخ الاحتياطي
src/utils/calculations.js       معادلات الحساب
src/context/AppContext.jsx      الحفظ والثيم والبيانات
public/manifest.webmanifest     إعدادات تثبيت PWA
public/sw.js                    العمل دون اتصال
```
